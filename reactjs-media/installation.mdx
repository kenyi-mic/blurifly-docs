## Installation

To install `reactjs-media` you need to run this command in the terminal

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs
  defaultValue="npm"
  values={[

  ]

<TabItem value="npm">

```bash
npm install reactjs-media
```

</TabItem>
<TabItem value="yarn">

```bash
yarn add reactjs-media
```

</TabItem>

</Tabs>

:::warning

When installing reactjs-media we recommend installing a version that is greater then `v1.1.2` since you might come across errors most especially if you are using server side rendering. Its also better to usally install the latest versions.

:::

## Install latest version

Inoder to install the latest version of any package from npm add the `@latest` tag on the install command.

```bash
npm install reactjs-media@latest
```
